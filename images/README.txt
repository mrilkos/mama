📷 HOW TO ADD YOUR PHOTOS
───────────────────────────
Drop your 3 photos in this folder and name them:

  photo1.jpg  ←  Baby / toddler photo
  photo2.jpg  ←  Child / teen photo
  photo3.jpg  ←  Recent / adult photo

Recommended: portrait orientation, at least 600×750px.
Any common format works: .jpg .jpeg .png .webp

That's it — no code changes needed!
